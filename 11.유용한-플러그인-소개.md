# 11단계: 유용한 플러그인 소개

> 이 챕터에서 할 것: Claude Code를 더욱 강력하게 만드는 플러그인과 확장 기능들을 소개합니다.

---

## 11-1. 두 가지 확장 방식

Claude Code는 두 가지 방식으로 기능을 확장할 수 있습니다.

| | Claude Code 플러그인 | MCP 서버 |
|---|---|---|
| **역할** | AI의 작업 방식을 개선 | 외부 도구와 연결 |
| **예시** | Superpowers (브레인스토밍, 디버깅 등) | GitHub, 웹 검색, 데이터베이스 |

---

## 11-2. Superpowers 플러그인

**Superpowers**는 Claude Code의 작업 방식을 체계적으로 만들어주는 **Claude Code 플러그인**입니다.  
MCP 서버가 아니라, Claude Code 내부에서 동작하는 스킬(skill) 모음입니다.

주요 기능:
- **브레인스토밍** — 새로운 기능을 만들기 전에 AI와 함께 아이디어를 구체화
- **체계적인 디버깅** — 버그를 단계별로 분석하고 해결
- **테스트 주도 개발** — 코드 작성 전에 테스트를 먼저 작성하는 방식으로 안정적인 코드 유지

설치 방법:

1. 터미널에서 Claude Code 실행
```bash
claude
```

2. Claude Code 세션 안에서 슬래시 명령어 입력
```
/plugin
```

3. 플러그인 매니저 UI에서 **superpowers** 검색 후 설치

---

## 11-3. MCP 서버란?

**MCP(Model Context Protocol)**는 Claude Code가 외부 도구와 대화하는 방식입니다.  
MCP 서버를 추가하면 Claude Code가 GitHub에서 코드를 가져오거나, 웹을 검색하거나, 데이터베이스를 조회할 수 있게 됩니다.

---

## 11-4. 유용한 MCP 서버 목록

### GitHub MCP
GitHub 저장소와 연결해서 코드 관리를 Claude Code에서 바로 할 수 있습니다.

```bash
claude mcp add github
```

활용 예시:
```
내 GitHub에 새 저장소를 만들어줘
이 코드를 GitHub에 올려줘
```

---

### Brave Search MCP
Claude Code에서 웹 검색을 바로 할 수 있습니다.

```bash
claude mcp add brave-search
```

활용 예시:
```
최신 CSS 트렌드를 검색해서 내 앱 디자인에 적용해줘
```

---

### SQLite MCP
데이터베이스와 연결해서 데이터를 저장하고 조회할 수 있습니다.

```bash
claude mcp add sqlite
```

활용 예시:
```
할 일 목록 데이터를 데이터베이스에 저장하도록 앱을 수정해줘
```

---

## 11-5. 설치된 플러그인 확인하기

현재 설치된 MCP 서버 목록을 확인하려면:

```bash
claude mcp list
```
