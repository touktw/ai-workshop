# 🖥️ Cursor로 시작하는 웹 개발 입문

## 소개

개발 지식이 전혀 없어도 괜찮습니다.  
이 가이드는 Cursor AI 코드 에디터를 활용해서  
**맥북 환경 세팅부터 첫 번째 웹앱(할 일 목록) 완성까지** 단계별로 안내합니다.

Cursor의 AI 기능 덕분에 코드를 직접 작성하지 않아도 됩니다.  
AI에게 원하는 것을 설명하면, AI가 코드를 만들어 줍니다.

## AI와 함께 개발하는 마음가짐

AI 도구를 처음 쓰는 사람들이 가장 많이 하는 실수는, **기존 방식 그대로 AI를 끼워 넣으려는 것**입니다.

코드를 직접 짜던 사람은 AI에게 한 줄씩 물어봅니다.  
구글에 검색하던 사람은 AI에게 검색하듯 짧게 질문합니다.  
혼자 고민하던 사람은 AI가 있어도 혼자 고민합니다.

AI는 기존 방식을 빠르게 해주는 도구가 아닙니다.  
**일하는 방식 자체를 바꿔야 AI의 진짜 가능성이 열립니다.**

---

**틀을 바꾸는 세 가지 관점**

**1. 완성보다 방향이 먼저다**  
AI는 코드를 빠르게 만들어 줍니다. 그래서 "어떻게 만들까"보다 "무엇을 만들까"가 훨씬 중요해졌습니다. 방향이 틀리면 AI는 틀린 것을 빠르게 만들어 줄 뿐입니다.

**2. 모르는 것을 부끄러워하지 마라**  
AI 앞에서는 전문가처럼 질문할 필요가 없습니다. 모르면 모른다고, 어색하면 어색하다고 그대로 말하세요. AI는 맥락을 이해하고 수준에 맞게 답합니다. 오히려 솔직한 질문이 더 좋은 답을 만듭니다.

**3. 빠르게 틀리고, 빠르게 고쳐라**  
완벽한 계획을 세우는 데 시간을 쓰지 마세요. AI와 함께라면 실험의 속도가 달라집니다. 일단 만들어 보고, 이상하면 고치면 됩니다. 두려움 없이 시도하는 것이 AI 시대의 핵심 역량입니다.

---

이 가이드는 코드를 가르치지 않습니다.  
**AI와 함께 생각하고, 만들고, 고치는 경험**을 안내합니다.
